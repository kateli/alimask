alimask

Project alimask is a simple library to generate water mask with canvas used in alibaba / antfin. 用于生成阿里内部网站水印的 JavaScript 库。

Install 

npm install alimask

Then import it. 

// import library use script tag.
<script type="text/javascript" src="dist/alimask.min.js"></script>

// or
var alimask = require('alimask');

// or
import alimask from 'alimask';

Usage

var base64 = alimask('王小为(小为) 888888');
